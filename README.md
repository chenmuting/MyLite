# MyLite

## 应用详情

一款轻量实用的鸿蒙版 MySQL 连接工具，帮你轻松搞定跨设备数据库管理。

### 主要特性



*   支持自行搭建后端服务器，连接 MySQL 数据库

*   支持手机、平板、电脑等鸿蒙设备

*   操作简单，无需复杂设置，上手即用
